Property Description Possible Values

Must use the american spelling		 Predefined colours, Hex, RGB
text-align Changes the justification of text Left, Right, Cener, Justify
background-color Changes the color of the background

font-family Changes the typeface of an element Helvetica, Georgia, Arial
font-weight Changes the thickness of font in an element Normal, Bold, lighter
